# PhanFlix - Phan Viet Portfolio (Refactored)

This is a static website clone of phanviet.me, refactored to match the exact design of the provided source code using **Tailwind CSS**.

## Design System
- **Framework**: Tailwind CSS (via CDN).
- **Fonts**: Bebas Neue (Headings), Inter (Body).
- **Icons**: Lucide Icons.
- **Colors**: Netflix Red (`#E50914`), Dark Background (`#141414`).

## Features
- **Responsive Navbar**: Transparent to black gradient on scroll.
- **Hero Section**: Full-screen immersive background with "Phan Viet" branding.
- **Content Sliders**: Horizontal scrolling rows with hover-expand effects.
- **Data**: Rich content (Match %, Tags, Views) populated from source data.

## How to Run
1. Open `index.html` in any web browser.
2. Ensure you have an internet connection (to load Tailwind CDN and Images).
